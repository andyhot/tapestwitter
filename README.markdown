TapesTwitter
============

#### A Tapestry 5 demo application ####


TapesTwitter is a "Twitter-like" demonstration application build with Tapestry 
5.

Build around :

* Tapestry
* Spring (IoC and Spring Security)
* JPA (with Hibernate implementation)
* HSQL
* Unitils
